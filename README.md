# Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript and hosted on GitHub Pages.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

## Project Structure

```text
portfolio/
├── index.html      # Main webpage structure
├── styles.css      # Website styling and responsive design
└── script.js       # Interactive functionality
```

## Development Environment

* Visual Studio Code
* Live Server Extension
* Git
* GitHub

## Running Locally

Clone the repository:

```bash
git clone https://github.com/hemanth-571/portfolio.git
```

Navigate to the project folder:

```bash
cd portfolio
```

Open the project in Visual Studio Code and launch `index.html` using the Live Server extension.

## Deployment

This project is deployed using GitHub Pages.

To publish updates:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

GitHub Pages automatically redeploys the latest version after changes are pushed to the repository.
